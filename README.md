## 它到底是怎么用的？

鼠标移到屏幕顶部刘海区域，Folder Hub 的浮窗就会出现。移开鼠标，浮窗自动隐藏。这个设计很聪明——既不占用菜单栏图标位，也不会莫名其妙遮住你正在做的事。

如果你用的是没有刘海的旧款 Mac，或者在全屏模式下工作，可以用快捷键 `⌘ + Option + →` 在鼠标位置呼出浮窗，效果一样。

浮窗里可以创建「工作空间」，本质上就是你自定义的文件夹快捷方式集合。比如建一个「设计素材」工作空间放着常用图片文件夹，建一个「当前项目」放着开发目录——鼠标在不同工作空间名字上滑一下，内容立刻切换。

文件操作方面，和 Finder 高度一致：空格键预览文件、双击进入子文件夹、右键选择打开方式、直接拖拽到其他窗口……基本没有学习成本。

支持多显示器——鼠标在哪块屏幕，浮窗就在哪块屏幕出现，这个细节很舒服。
